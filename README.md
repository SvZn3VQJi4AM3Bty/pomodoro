# Okonai (GitHub Pages)

GitHub Pages で動く、静的ファイル（HTML/CSS/JSのみ）のカレンダーとポモドーロタイマーです。

## 主な機能

- ポモドーロタイマー（没入 / 小休憩 / 長休憩）
- 円形の進捗表示（モードごとに配色が変化）
- 残り時間（数値）表示の ON/OFF
- 現在日時表示の ON/OFF
- 月移動できるカレンダー（前月 / 次月）
- 土日と日本の祝日の表示切り分け
- 終了音のパターン選択（無音を含む）と音量調整
- 設定の自動保存（localStorage）

## 使い方（ローカル）

このプロジェクトはビルド不要です。

- `index.html` をブラウザで開く
- もしくは簡易サーバーで起動（例: VS Code / Cursor の Live Server など）

## 画面上の主な操作

- ヘッダー右上
  - `残り時間`：円形タイマー中央の数値表示を切り替え
  - `現在日時`：現在日時カードとカレンダー表示を切り替え
- タイマーセクション下
  - `設定と使い方`：設定セクションと使い方セクションをまとめて表示/非表示
- 設定セクション
  - 変更はすべて自動保存（保存ボタン不要）
  - `終了音を試聴` は現在フォームの選択内容（パターン・音量）で再生

## GitHub Pages で公開

1. GitHub のリポジトリを開く
2. Settings → Pages
3. Build and deployment → Source を **Deploy from a branch** に設定
4. Branch を `main` / `/(root)` に設定して保存

しばらくすると `https://<user>.github.io/<repo>/` で公開されます。

## スマホで「ホーム画面に追加」（PWA）

このページは PWA 対応なので、スマホのホーム画面に追加してアプリのように使えます。

### iPhone / iPad（Safari）

1. **Safari** でページを開く
2. 画面下の **共有ボタン（四角＋↑）** をタップ
3. **「ホーム画面に追加」** を選ぶ
4. 名前を確認して **「追加」**

### Android（Chrome）

1. **Chrome** でページを開く
2. 右上の **︙（メニュー）** をタップ
3. **「ホーム画面に追加」** または **「アプリをインストール」** を選ぶ
4. 表示に従って追加/インストール

※ 追加/インストールが出ない場合は、数秒待つ・再読み込みする・HTTPSで開いているか確認してください。

## 「できるだけ知られない」設定について（重要）

GitHub Pages は **公開Webサイト**なので、URLを知っている人はアクセスできます（完全な非公開にはできません）。
それでも「検索に出にくくする」ために、このプロジェクトでは以下を設定しています。

- `robots.txt` でクローラーを拒否
- `index.html` に `noindex`（検索結果に出さない）を指定

より確実に「第三者に見られない」ようにしたい場合は、次のどれかが必要です。

- **リポジトリを private にして Pages を使う**（GitHub のプラン/設定によって可否が変わります）
- **認証付きで配信**（Cloudflare Access / Basic認証できるホスティング等）
- **ローカル専用で使う**（端末内だけで完結）

## セキュリティ（最低限）

- 設定は `localStorage` に保存され、サーバーへ送信しません
- 外部ライブラリ依存なし（サプライチェーンリスクを小さく）

## AI実装ポリシー（外部通信禁止）

このプロジェクトでは、AIによる実装時も次を禁止します。

- 外部サービスへの通信を伴う機能追加（API連携、分析タグ、広告、トラッキング等）
- 外部通信を行うSDK/ライブラリの導入

例外として、PWA更新維持に必要な同一オリジンのService Worker更新確認のみ許可します。
